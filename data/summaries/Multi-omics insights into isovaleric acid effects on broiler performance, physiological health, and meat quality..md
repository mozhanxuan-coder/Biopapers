# Multi-omics insights into isovaleric acid effects on broiler performance, physiological health, and meat quality.

> **PMID**: 42321165 | **Journal**: NPJ biofilms and microbiomes
> **Link**: [点击直达 PubMed 论文页面](https://pubmed.ncbi.nlm.nih.gov/42321165/)

---

【研究背景与动机】
家禽养殖业中，饲料添加剂对肉鸡生长性能、生理健康及肉品质的调控作用一直是研究热点。异戊酸（Isovaleric acid, IVA）作为一种短链脂肪酸，已被证实对肠道健康具有积极影响，但其对肉鸡肉品质的具体作用机制尚不明确。现有研究多聚焦于IVA对肠道微生物群落的调节，而缺乏从多组学层面系统解析IVA如何通过“肠道微生物-代谢物-肌肉”轴影响肉品质的整合性证据。因此，本研究旨在通过多组学方法（微生物组、代谢组、血液生化及肉品质指标）全面评估IVA对肉鸡生产性能、生理状态及肉品质的剂量依赖性效应，并揭示其潜在分子机制，为家禽营养干预策略提供科学依据。

【详尽研究方法】
本研究采用随机对照实验设计，共纳入864只肉鸡，随机分为4组：对照组（基础日粮）及3个IVA处理组（分别添加0.05%、0.1%和0.2%的IVA）。实验周期为42天，在21天和42天两个时间点进行采样与分析。具体方法如下：
1. **生产性能评估**：记录平均日增重（ADG）、采食量（FI）及饲料转化率（FCR）。
2. **血液生化与免疫指标**：检测血清中血尿素氮（BUN）、炎症因子（如IL-6、TNF-α）及免疫球蛋白（IgA、IgG）水平。
3. **肠道形态学**：测量十二指肠、空肠和回肠的绒毛高度（VH）及隐窝深度（CD），计算VH/CD比值。
4. **肉品质分析**：于42天采集胸肌和腿肌样本，测定蛋白质含量、滴水损失（drip loss）、剪切力（shear force）及色泽参数（L*、a*、b*值）。
5. **微生物组分析**：提取盲肠内容物DNA，对细菌16S rRNA基因V3-V4区及真菌ITS1区进行高通量测序。使用QIIME2流程进行序列处理，计算α多样性（Ace指数、Shannon指数）和β多样性（基于Bray-Curtis距离的PCoA分析）。采用LEfSe分析鉴定差异丰度菌属。
6. **代谢组学分析**：采用超高效液相色谱-串联质谱（UPLC-MS/MS）对盲肠内容物及血清进行非靶向代谢组学检测。通过主成分分析（PCA）和正交偏最小二乘判别分析（OPLS-DA）筛选差异代谢物，并利用KEGG数据库进行通路富集分析。
7. **整合分析**：采用Spearman相关性分析，将差异菌属与差异代谢物及肉品质指标进行关联，构建“肠道微生物-代谢物-肌肉”调控网络。

【核心数据与结果】
1. **生产性能**：与对照组相比，0.1%和0.2% IVA组肉鸡的ADG显著提高（P < 0.05），其中0.1% IVA组ADG提升约8.2%。0.05%和0.1% IVA组FI增加（P < 0.05），而0.2% IVA组FCR降低约5.6%（P < 0.05）。
2. **血液生化与免疫**：21天和42天时，所有IVA处理组血清BUN水平均显著低于对照组（P < 0.05），降幅达12-18%。0.1% IVA组血清IL-6水平降低约22%（P < 0.05），而IgA和IgG水平分别升高15%和11%（P < 0.05）。
3. **肠道形态**：0.1%和0.2% IVA组十二指肠和空肠的VH显著增加（P < 0.05），增幅为10-15%，VH/CD比值亦显著升高（P < 0.05）。
4. **肉品质**：42天时，0.1%和0.2% IVA组胸肌和腿肌蛋白质含量分别增加6.3%和7.8%（P < 0.05）。滴水损失降低18-25%（P < 0.05），剪切力降低12-16%（P < 0.05）。色泽参数中，a*值（红度）显著升高（P < 0.05），b*值（黄度）降低（P < 0.05）。
5. **微生物组**：21天时，IVA处理组细菌Ace指数显著降低（P < 0.05），β多样性发生显著改变（PERMANOVA，P < 0.01）。42天时，0.1% IVA组细菌Shannon指数显著升高（P < 0.05），而真菌α多样性无显著变化。LEfSe分析显示，Collinsella、norank_f__Ruminococcaceae及unclassified_f__Oxalobacteraceae在IVA组中富集（LDA score > 2.0）。
6. **代谢组学**：共鉴定出127种差异代谢物，主要富集于脂质代谢（如甘油磷脂代谢、鞘脂代谢）和氨基酸代谢（如精氨酸与脯氨酸代谢、色氨酸代谢）通路。其中，与肌肉蛋白合成相关的代谢物（如肌酸、牛磺酸）在IVA组中显著上调（VIP > 1.0，P < 0.05）。
7. **整合分析**：Spearman相关性分析表明，Collinsella丰度与肌酸水平呈正相关（r = 0.62，P < 0.01），与滴水损失呈负相关（r = -0.55，P < 0.05）；norank_f__Ruminococcaceae丰度与牛磺酸水平呈正相关（r = 0.58，P < 0.01），与剪切力呈负相关（r = -0.51，P < 0.05）。这些结果支持“肠道微生物-代谢物-肌肉”轴的存在。

【领域应用与启示】
本研究首次通过多组学整合分析，系统揭示了IVA通过调节肠道微生物组成（尤其是Collinsella、Ruminococcaceae等关键菌属）及宿主代谢网络（脂质与氨基酸代谢），进而改善肉鸡生长性能、生理健康及肉品质的分子机制。该成果为家禽生产中利用短链脂肪酸作为功能性饲料添加剂提供了理论依据，尤其为通过靶向肠道微生物群来提升肉品质（如降低滴水损失、改善嫩度及色泽）提供了可操作的营养干预策略。未来研究可进一步探索IVA与其他益生元或益生菌的协同效应，并利用无菌动物模型或粪菌移植实验验证关键菌属的因果作用。此外，该研究建立的“微生物-代谢物-表型”关联分析框架，可推广至其他家畜或水产养殖领域，用于评估新型饲料添加剂的功能机制。
