# Stage-specific rhizosphere microbial succession is associated with nutrient cycling in the desert plant Leymus racemosus (Lam.) tzvelev.

> **PMID**: 42321605 | **Journal**: BMC plant biology
> **Link**: [点击直达 PubMed 论文页面](https://pubmed.ncbi.nlm.nih.gov/42321605/)

---

【研究背景与动机】
在荒漠生态系统中，植物生长受到养分（如碳、氮、磷、钾）可利用性的严格限制。植物根系通过分泌特定根际分泌物，招募根际微生物群落，从而调节养分循环与吸收。然而，现有研究多集中于单一生长阶段或特定养分元素，缺乏对根际核心微生物组与功能微生物组在整个植物生命周期中如何协同调控碳、氮、磷、钾动态的系统性理解。特别是对于荒漠植物赖草（*Leymus racemosus*），其根际细菌群落的演替规律是否与不同生长阶段的养分需求相匹配，仍是一个未解决的关键科学问题。本研究旨在揭示赖草根际细菌群落的时序演替模式，并阐明其与养分循环功能之间的关联机制。

【详尽研究方法】
本研究采用田间实验设计，在荒漠生态系统中选取赖草作为研究对象，覆盖其完整生命周期（包括幼苗期、营养生长期、生殖生长期及衰老期）。实验设置多个重复样地，每个样地内随机采集根际土壤样品（紧贴根表的土壤，定义为根际）及非根际土壤（作为对照）。样品采集后，立即进行以下处理流程：
1. **土壤理化性质分析**：测定土壤pH、有机碳（SOC）、全氮（TN）、全磷（TP）、全钾（TK）、速效氮（AN）、速效磷（AP）、速效钾（AK）含量，以及土壤含水量和电导率。
2. **微生物DNA提取与扩增子测序**：使用PowerSoil DNA提取试剂盒提取土壤总DNA。针对细菌16S rRNA基因的V3-V4高变区进行PCR扩增，引物为338F（5‘-ACTCCTACGGGAGGCAGCA-3’）和806R（5‘-GGACTACHVGGGTWTCTAAT-3’）。扩增产物经纯化、定量后，在Illumina MiSeq平台进行双端测序（2×300 bp）。
3. **生物信息学分析流程**：
   - 使用QIIME2（版本2020.11）对原始序列进行质控、去噪（DADA2算法）、拼接，并去除嵌合体，生成扩增子序列变体（ASV）表。
   - 基于SILVA数据库（版本138）对ASV进行物种注释。
   - 计算α多样性指数（Shannon指数、Chao1指数）和β多样性（基于Bray-Curtis距离的主坐标分析，PCoA）。
   - 使用LEfSe分析（线性判别分析效应大小）鉴定不同生长阶段根际与非根际土壤中的差异丰度细菌类群。
   - 通过PICRUSt2（系统发育调查中的群落重建）基于16S rRNA基因序列预测细菌群落的潜在功能基因丰度，重点关注与碳（如纤维素降解、甲烷代谢）、氮（如固氮、硝化、反硝化）、磷（如有机磷矿化、无机磷溶解）和钾（如钾溶解）循环相关的KEGG通路。
4. **核心微生物组鉴定**：使用网络分析（基于Spearman相关性，p < 0.05，|r| > 0.6）构建根际细菌共现网络，并利用Zi-Pi（模块内连接度与模块间连接度）图识别核心微生物（包括模块枢纽和连接器）。
5. **统计建模**：采用冗余分析（RDA）和方差分解分析（VPA）评估土壤理化因子与微生物群落结构之间的关系。使用Mantel检验验证微生物群落演替与养分动态之间的相关性。

【核心数据与结果】
1. **根际细菌群落演替模式**：PCoA分析显示，根际细菌群落结构在赖草不同生长阶段呈现显著分离（PERMANOVA，R² = 0.42，p < 0.001），而非根际土壤群落结构随时间变化不显著（p > 0.05）。这表明植物发育阶段是驱动根际微生物群落演替的主要因素。
2. **核心微生物组动态**：网络分析共鉴定出12个核心ASV，主要归属于放线菌门（*Actinobacteria*）、变形菌门（*Proteobacteria*）和厚壁菌门（*Firmicutes*）。这些核心ASV的相对丰度随生长阶段显著变化：例如，在营养生长期，*Streptomyces*（链霉菌属）和*Bacillus*（芽孢杆菌属）的相对丰度达到峰值（分别占根际细菌总数的8.2%和5.6%），而在衰老期则显著下降（分别降至2.1%和1.3%，p < 0.01）。
3. **养分循环功能预测**：PICRUSt2预测结果显示，与碳循环相关的功能基因（如编码纤维素酶、β-葡萄糖苷酶的基因）在营养生长期和生殖生长期的根际中显著富集（丰度较幼苗期增加2.3倍，p < 0.05）。与氮循环相关的固氮基因（*nifH*）在营养生长期丰度最高（较幼苗期增加1.8倍，p < 0.05），而硝化基因（*amoA*）在生殖生长期达到峰值。与磷循环相关的有机磷矿化基因（*phoD*）在生殖生长期和衰老期显著上调（丰度较幼苗期增加3.1倍，p < 0.01）。与钾循环相关的钾溶解基因（如编码钾转运蛋白的基因）在营养生长期丰度最高。
4. **养分动态与微生物关联**：RDA分析表明，土壤速效氮（AN）、速效磷（AP）和速效钾（AK）含量解释了根际细菌群落变异的38.7%（p < 0.001）。Mantel检验进一步证实，根际细菌群落结构与土壤AN（r = 0.61，p < 0.001）、AP（r = 0.55，p < 0.001）和AK（r = 0.48，p < 0.01）含量显著相关。具体而言，在营养生长期，根际AN和AK含量显著高于非根际土壤（分别高出32%和27%，p < 0.05），这与固氮和钾溶解功能基因的富集一致。

【领域应用与启示】
本研究首次系统揭示了荒漠植物赖草根际细菌群落在完整生命周期中的时序演替规律，并定量证明了该演替与植物阶段性养分需求（尤其是氮、磷、钾）的耦合关系。其核心贡献在于：1）明确了核心微生物组（如*Streptomyces*和*Bacillus*）在关键养分循环中的潜在功能角色，为开发基于微生物的荒漠土壤改良剂提供了靶标菌株；2）建立了植物发育阶段与根际微生物功能基因丰度之间的直接关联，为预测荒漠生态系统养分动态提供了微生物学指标；3）强调了在生态修复实践中，应根据植物生长阶段动态调整微生物接种策略（例如，在营养生长期优先接种固氮和钾溶解菌，在生殖生长期优先接种解磷菌）。未来研究可进一步结合宏转录组或宏蛋白组技术，直接验证预测功能基因的实际表达活性，并探索这些核心微生物与植物激素信号（如独脚金内酯、水杨酸）之间的分子互作机制，以深化对植物-微生物协同适应极端环境的理解。
