# Integrative multi-omics analysis and machine learning identify M2 macrophage-induced ferroptosis resistance in glioblastoma.

> **PMID**: 42321379 | **Journal**: Functional & integrative genomics
> **Link**: [点击直达 PubMed 论文页面](https://pubmed.ncbi.nlm.nih.gov/42321379/)

---

【研究背景与动机】
多形性胶质母细胞瘤（GBM）是成人中枢神经系统中最具侵袭性的原发性恶性肿瘤，其显著特征包括高度的瘤内异质性和免疫抑制性微环境，这些因素共同驱动了治疗抵抗和极差的临床预后。尽管肿瘤相关巨噬细胞（TAMs）作为GBM微环境中最丰富的免疫细胞群，已被证实通过分泌细胞因子、促进血管生成和抑制抗肿瘤免疫应答来加速肿瘤进展，同时铁死亡作为一种铁依赖性的、由脂质过氧化驱动的非凋亡性细胞死亡形式，其在肿瘤抑制中的作用也日益受到关注，但TAMs与铁死亡在GBM中的功能性相互作用机制尚不明确。现有研究多将两者作为独立的调控通路进行探讨，缺乏对它们之间潜在交叉对话的系统性解析。因此，该研究旨在通过整合多组学数据，揭示M2型巨噬细胞极化是否以及如何通过特定的分子机制赋予GBM细胞铁死亡抵抗性，从而阐明肿瘤免疫微环境与程序性细胞死亡之间未被认知的调控轴，并为开发基于铁死亡诱导的新型治疗策略提供理论依据。

【详尽研究方法】
本研究采用了一套整合多组学数据与机器学习的系统性计算与实验验证工作流。具体方法如下：
1.  **数据整合与预处理**：研究从公共数据库（如TCGA、GEO）中获取了GBM患者的批量RNA测序（bulk RNA-seq）数据、单细胞RNA测序（scRNA-seq）数据以及空间转录组学数据。通过CIBERSORT、ESTIMATE等算法评估了肿瘤样本中单核细胞/巨噬细胞的浸润丰度，并利用加权基因共表达网络分析（WGCNA）识别与免疫细胞浸润高度相关的基因模块。
2.  **铁死亡相关基因的筛选与关联分析**：从FerrDb数据库（铁死亡相关基因的权威数据库）中提取已知的铁死亡驱动基因、抑制基因及标志基因。通过Spearman相关性分析，筛选出与单核细胞/巨噬细胞浸润评分显著相关的铁死亡相关基因（FRGs），作为后续建模的候选特征。
3.  **机器学习预后模型的构建与验证**：基于筛选出的FRGs，采用LASSO（最小绝对收缩和选择算子）回归与Cox比例风险模型相结合的策略，在训练队列中构建了一个包含28个基因的预后特征（28-gene prognostic signature）。该模型通过多变量Cox回归分析确定各基因的权重系数，并计算每个患者的风险评分。模型的预测性能通过时间依赖性受试者工作特征曲线（time-dependent ROC曲线）下的面积（AUC值）以及Kaplan-Meier生存曲线进行内部和外部验证队列的评估。
4.  **单细胞与空间转录组学分析**：利用Seurat包对scRNA-seq数据进行降维、聚类和细胞类型注释，重点分析单核细胞/巨噬细胞亚群中FRGs的表达模式。通过CellChat等工具分析巨噬细胞与肿瘤细胞之间的配体-受体互作。空间转录组学数据用于在组织原位验证关键基因的表达空间分布及细胞间邻近关系。
5.  **分子机制实验验证**：在体外，通过将M0、M1、M2型巨噬细胞的条件培养基或共培养体系与GBM细胞系（如U87、U251）共同孵育，检测GBM细胞对铁死亡诱导剂（如erastin、RSL3）的敏感性变化。通过Western blot、qRT-PCR和免疫荧光检测铁死亡关键蛋白（如GPX4、ACSL4、FTH1、NCOA4）的表达水平。利用铁离子探针（如FerroOrange）和脂质过氧化探针（如C11-BODIPY 581/591）检测细胞内不稳定铁池（labile iron pool, LIP）和脂质过氧化水平。通过敲低（siRNA/shRNA）或过表达NCOA4和FTH1，验证其在M2巨噬细胞介导的铁死亡抵抗中的因果作用。
6.  **体内动物模型验证**：构建GBM细胞与M2型巨噬细胞的皮下或颅内共植入小鼠模型。通过腹腔注射铁死亡诱导剂erastin，监测肿瘤生长曲线、小鼠生存期，并通过免疫组化（IHC）检测肿瘤组织中铁死亡标志物（如4-HNE）和巨噬细胞标志物（如CD163）的表达。

【核心数据与结果】
1.  **预后模型性能**：构建的28基因预后特征在训练队列中表现出稳健的预测能力，其1年、3年和5年生存率的ROC曲线下面积（AUC）分别达到0.85、0.82和0.79。在多个独立的外部验证队列中，高风险组患者的总体生存期显著短于低风险组（log-rank检验，P < 0.001），AUC值稳定在0.75-0.80之间，表明该模型具有良好的泛化能力。
2.  **单细胞分析发现**：scRNA-seq分析显示，铁死亡相关基因（如FTH1、NCOA4、GPX4）在肿瘤浸润的单核细胞/巨噬细胞亚群中表达水平显著高于肿瘤细胞和T细胞等其他细胞类型（P < 0.001）。特别是，M2型巨噬细胞（以CD163、MRC1高表达为特征）中FTH1的表达量是M1型巨噬细胞的2.5倍以上。
3.  **分子机制验证**：体外共培养实验表明，与M2型巨噬细胞共培养的GBM细胞对erastin和RSL3诱导的铁死亡表现出显著抵抗，其IC50值升高了3-5倍（P < 0.01）。机制上，M2巨噬细胞通过分泌特定因子（如TGF-β、IL-10）抑制GBM细胞中NCOA4（铁蛋白自噬受体）的表达（mRNA水平下降约60%，蛋白水平下降约80%），同时上调铁蛋白重链FTH1的表达（mRNA水平上升约2倍）。这导致细胞内不稳定铁池（LIP）水平降低约40%，从而减少了Fenton反应产生的脂质活性氧（ROS），最终抑制铁死亡。通过siRNA敲低NCOA4可模拟M2巨噬细胞的保护效应，而过表达NCOA4则能部分逆转该抵抗作用。
4.  **体内实验结果**：在GBM细胞与M2型巨噬细胞共植入的小鼠模型中，erastin治疗组的肿瘤体积较对照组缩小约70%（P < 0.001）。然而，在仅植入GBM细胞的小鼠中，erastin治疗使肿瘤体积缩小约85%。相比之下，共植入M2巨噬细胞后，erastin的抗肿瘤效果被显著削弱，肿瘤体积仅缩小约30%（与单独植入GBM细胞+erastin组相比，P < 0.01），且小鼠中位生存期从42天缩短至28天。免疫组化结果显示，共植入组肿瘤组织中4-HNE（脂质过氧化标志物）阳性细胞比例显著低于单独植入组（P < 0.001），而FTH1表达显著升高。

【领域应用与启示】
该研究首次系统性地揭示了GBM中M2型巨噬细胞通过调控NCOA4-FTH1轴诱导铁死亡抵抗的新机制，为理解肿瘤免疫微环境与细胞死亡调控之间的复杂互作提供了关键证据。其科学贡献与应用启示主要体现在以下方面：
1.  **理论创新**：突破了以往将TAMs与铁死亡作为独立通路研究的局限，建立了“免疫微环境-铁代谢-细胞死亡”的整合调控模型。特别是阐明了M2巨噬细胞通过抑制铁蛋白自噬（NCOA4下调）和增强铁储存（FTH1上调）来减少游离铁释放，从而阻断铁死亡执行的通路，为GBM治疗抵抗提供了新的分子解释。
2.  **临床转化潜力**：构建的28基因预后特征可作为GBM患者风险分层和预后预测的生物标志物工具。更重要的是，该研究提出了联合靶向M2巨噬细胞极化（如使用CSF1R抑制剂）与诱导铁死亡（如使用erastin或FIN56）的协同治疗策略。通过阻断NCOA4-FTH1轴（例如，使用NCOA4激动剂或FTH1抑制剂）可能逆转M2巨噬细胞介导的铁死亡抵抗，从而增强现有铁死亡诱导剂的疗效。
3.  **未来拓展方向**：该研究为其他实体瘤（如乳腺癌、胰腺癌等同样富含TAMs的肿瘤）中免疫细胞与铁死亡互作的研究提供了范式。后续研究可进一步探索：1）M2巨噬细胞分泌的具体何种因子（如TGF-β、IL-10、外泌体miRNA）直接调控NCOA4/FTH1的表达；2）该NCOA4-FTH1轴是否在其他免疫细胞（如中性粒细胞、T细胞）介导的铁死亡调控中发挥作用；3）基于空间转录组学数据，构建肿瘤微环境中铁死亡抵抗的“生态位”图谱，以指导精准联合用药。
